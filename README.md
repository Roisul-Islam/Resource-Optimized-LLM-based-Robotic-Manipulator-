# Resource-Optimized-LLM-based-Robotic-Manipulator-
An intelligent sorting robot system built for EEE 404 (Robotics & Automation Lab) at BUET. The system combines lightweight computer vision (OpenCV) with LLM-based reasoning (LLaMA 3.2) to interpret natural-language instructions and autonomously sort objects using a 6DOF robotic arm controlled via Arduino Uno.
Achieves 95%+ detection and sorting accuracy on standard laptop hardware — no GPU or expensive industrial infrastructure needed.
